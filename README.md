# Лабораторна робота №4

## Тема
Контейнеризація сервісу (Docker)

## Модель
Туризм: Розподіл завантаження номерів у готелі (на прикладі Marriott, Hyatt)

## Автор
Антонов В. В., група АІ-235 (Варіант №1, MODE=comfort)

## Опис
Проєкт містить контейнеризовану за допомогою Docker математичну модель чисельної оптимізації рівня овердокінгу в готелях. Програма реалізована на мові Python з використанням бібліотеки NumPy та розгорнута у вигляді автономного вебсервера всередині легковагого контейнера `python:3.10-slim`. Вона динамічно зчитує змінні середовища (конфігурацію режиму `comfort` та дані автора) й повертає результати аналізу на порт 5000.
