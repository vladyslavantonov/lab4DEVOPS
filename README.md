# Лабораторна робота №3

## Тема
Git та Linux CLI

## Модель
Туризм: Розподіл завантаження номерів у готелі (на прикладі Marriott, Hyatt)

## Автор
Антонов В. В., група АІ-235

## Опис
Реалізація обчислювальної моделі з курсу «Обчислювальні методи».
Програма симулює випадковий попит на номери готелю та використовує чисельний метод градієнтного спуску для пошуку оптимального рівня овердокінгу (перебронювання), що мінімізує збитки від пустих місць та штрафи за відмови клієнтам у заселенні.
Код написаний на Python.

## Як запустити
python main.py
